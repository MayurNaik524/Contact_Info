# Contact_Info
<h3>Running Backend</h3>
<p>Add port number and mongodb url in .env file</p>
<p>Intitialization of Backend</p>
<ul>
  <li>cd backend</li>
  <li>npm install</li>
</ul>
<p>To Run The Backend</p>
<ul>
  <li>npm start</li>
</ul><br/>
<h3>Running Frontend</h3>
<p>Initialization of Frontend</p>
<ul>
  <li>cd client</li>
  <li>npm install</li>
</ul>
<p>To Run The Frontend</p>
<ul>
  <li>npm run dev</li>
</ul>
