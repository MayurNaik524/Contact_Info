# Contact_Info
<h3>Running Backend</h3>
<p>Add port number and mongodb url in .env file</p>
<p>For Intitialization</p>
<ul>
  <li>cd backend</li>
  <li>npm install</li>
</ul>
<p>To Run The Backend</p><br/>
<ul>
  <li>npm start</li>
</ul><br/>

<p>Initialization of frontend</p>
<ul>
  <li>cd client</li>
  <li>npm install</li>
</ul>
<p>To Run The Frontend</p><br/>
<ul>
  <li>npm run dev</li>
</ul>
